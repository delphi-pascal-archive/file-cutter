При перемещении кусков никогда не забывайте головной файл - 'Head.vkq' !
-------------------------------------------------------------------------------------------------------------------------
Эта программа нужна для резки файлов на куски и для их последующего восстановления. Например,
вам нужно скопировать на дискету файл большого объёма, который даже в зазипованном виде на неё
не помещается. В этом случае можно разрезать файл на куски, а потом каждый из них скопировать на 
дискету. И всё! Правда, для того, чтобы склеить куски, вам понадобится ещё одна дискета для самой 
программы, но её объём не большой - 140 Кб. Если вы хотите перекинуть на дискеты (или куда-нибудь
ещё) сразу несколько файлов, то сначала их поместите в папку, папку зазипуйте, а потом уже ZIP-файл
режьте. При резке Cutter  не разрезает и не повреждает сам файл, а только делает копии его частей 
определённого объёма. Объём кусков задаётся пользователем. 
Для восстановления файла все его куски (*.vks), включая файл Head.vkq необходимо поместить в одну
папку, а потом уже всё это восстанавливать в программе.
